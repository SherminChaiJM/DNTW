This is the About Page for My Smart Home.

In this webpage, you will be able to create a smart environment for you home. Below, you will be able to find different help for the different functions available here.

Home

The Home page contains all the links to the different other pages.

About

The About page gives you more information to the rest of the pages in this web application. So feel free to come here to find more information, or to click on the header to go to the other pages.

Show Devices

The Show Devices page allow you to access all the devices that you have already added into the database. This allows you to keep track of the different devices in your home.

Add Devices

The Add Devices page allow you to add more devices whenever you get a new smart product. The page will allow you to customise the different setting for each products.

Delete Devices

The Delete Devices page allow you to delete any of the devices you have added to the database. This will completely remove the entry of the device.

Update Devices

The Update Devices page allow you to edit the information of the devices or adjust any settings of the devices. This comes in handy as you will be able to adjust everything through this web application instead of doing it manually on the devices.
